Neovim Shortcuts Cheatsheet

Leader: <leader> (según tu config) Modos: n normal · v visual · i insert Tip: imprime desde el navegador (Ctrl+P)

Telescope — Archivos & búsqueda finder / grep

ShortcutAcción
<leader>ffBuscar ficheros Git (si hay repo); si no, find_files con hidden.
<leader>faBuscar todos los ficheros (incluye hidden).
<leader>fgLive grep (excluye *.md).
<leader>psGrep string con prompt (input).
<leader>frTelescope resume (último picker).
Notas: tienes fd como find_command, ignores para node_modules, .next, etc.

Telescope — Buffers & ayuda navegación rápida

ShortcutAcción
<leader>fbBuffers.
<leader>fhHelp tags.
<leader>ftTreesitter (inspección/navegación).
<leader>mmMan pages.

Telescope — LSP símbolos / diagnósticos

ShortcutAcción
<leader>rrLSP references.
<leader>ttLSP type definitions.
<leader>ddDiagnostics (Telescope).
<leader>scDocument symbols: classes.
<leader>siDocument symbols: interfaces.
<leader>svDocument symbols: variables.
<leader>stDocument symbols: types + interfaces.

Telescope — Funciones (Tree-sitter) picker custom

ShortcutAcción
<leader>sf Lista de funciones coloreadas (export/named/arrow/method/callback) via Tree-sitter.
Incluye highlights: TSExportFunction, TSNamedFunction, TSArrowFunction, TSMethod, TSCallback.

UI — Colorscheme preview + transparencia

ShortcutAcción
<leader>cs Selector de colorscheme con preview; al confirmar aplica SetTransparentBackground().

Git (Telescope + custom) commits/branches/status

ShortcutAcción
<leader>gcGit commits (repo).
<leader>gccGit buffer commits con fecha (custom git log --date=short).
<leader>gbGit branches.
<leader>gsGit status.
<leader>gssArchivos distintos a la rama por defecto (usa git_branch'.files()) — mapeo en n y v.

General & edición productividad

ShortcutAcción
^Ir al buffer anterior (:b#).
<leader>pvAbrir explorer (:Ex).
v JMover selección visual 1 línea abajo.
v KMover selección visual 2 líneas arriba.
i Ctrl+cSalir a normal como Esc y volver al cursor anterior (<Esc>`^).
<leader>wGuardar todo (:wa).
Ctrl+rChecktime (:checktime) para recargar cambios externos.
<leader>wrkSwitchWorkspaceDropdown (función custom).

Rutas al clipboard copiar paths

ShortcutAcción
<leader>cppCopiar ruta absoluta del fichero actual al clipboard.
<leader>cpCopiar ruta relativa desde :pwd (valida que el archivo esté bajo el cwd).

LeetCode plugin commands

ShortcutAcción
<leader>lr:Leet run
<leader>lc:Leet console
<leader>ls:Leet submit
<leader>lm:Leet menu
<leader>lo:Leet open

Ejecutar fichero actual terminal split

ShortcutAcción
Ctrl+Enter Ejecuta el fichero actual en split con terminal, según extensión:
.c → gcc · .cpp → g++ -std=c++14 · .java → java · .py → python3 · .js → node · .ts → ts-node
